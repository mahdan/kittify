This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

## Download repository from github

```
git clone git@github.com:mahdan1075/kittify.git
```

## Go into directory

```
cd tailify-kittens
```

## Install packages

```
npm i
```

## Start website

```
npm start
```

## Start tests

```
npm test
```
